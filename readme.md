## プロジェクト概要
この Web システムは、個人がアクションゲーム「ELDEN RING NIGHTREIGN」を快適にプレイするための補助ツール

## 目的
- ゲームのプレイ中に必要なヒントをスマホから手早く取得する

## 想定ユーザー
- 個人
- スマートフォン利用を想定したGUI

## 主な機能
- ボス不明時、１日目、２日目ボス情報入力による３日目ボスの予想
    - ボスのパターンは`bosses.md`を参照
- 選択したボスの属性耐性、状態異常耐性を表示

## 技術スタック
- フロントエンド: HTML, CSS, JavaScript (純粋なクライアントサイドのみ、バックエンド不要)
- データ保存: localStorage (ブラウザローカル)
- デプロイ: GitHub Pages
- 公開環境: ブラウザ上の静的サイトとして動作
