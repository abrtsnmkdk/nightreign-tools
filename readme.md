## プロジェクト概要
この Web システムは、個人がアクションゲーム「ELDEN RING NIGHTREIGN」を快適にプレイするための補助ツール

## 目的
- ゲームのプレイ中に必要なヒントをスマホから手早く取得する

## 想定ユーザー
- 個人

## 主な機能
- ゲームの進行状況の入力による展開の予測
- MAPの表示
- 自分で残したメモ、攻略情報の閲覧

## 技術スタック
- フロントエンド: HTML, CSS, JavaScript (純粋なクライアントサイドのみ、バックエンド不要)
- データ保存: localStorage (ブラウザローカル)
- デプロイ: Google Driveにindex.htmlをアップロードし、共有リンクでモバイルアクセス

## 使い方
1. index.htmlをGoogle Driveにアップロード。
2. ファイルを右クリック > 「リンクを取得」 > 「リンクを知っている全員に変更」 > リンクを共有。
3. モバイルブラウザでリンクを開いて使用。
